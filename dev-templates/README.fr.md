## 💡 Pourquoi ce repo ?

À chaque nouveau projet web, on réécrit les mêmes composants : **pages légales**, **bannière cookies**, **footer**, **formulaires de contact**…

**dev-templates** résout ce problème. C'est une collection de templates **prêts à l'emploi**, testés en production, que vous pouvez copier dans n'importe quel projet Next.js et personnaliser en quelques minutes.

| | Avantage |
|:-:|----------|
| ⏱️ | **Gagnez du temps** — arrêtez de réécrire les mêmes pages pour chaque client |
| ✅ | **Qualité production** — chaque template a été utilisé dans des projets réels |
| 🌍 | **Open source** — utilisez-le gratuitement, contribuez si vous voulez, partagez-le |
| 🔤 | **Système de placeholders** — un simple rechercher-remplacer adapte tout à votre marque |

## 🚀 Quick Start

```bash
# 1️⃣  Clonez le repo
git clone https://github.com/LuigiGarone/open-source.git

# 2️⃣  Copiez le template dont vous avez besoin
cp -r open-source/dev-templates/legal-pages/components/ votre-projet/src/components/marketing/
cp -r open-source/dev-templates/legal-pages/pages/       votre-projet/src/app/(marketing)/

# 3️⃣  Remplacez les placeholders (voir le README de chaque template)
```
