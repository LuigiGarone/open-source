## 💡 Warum dieses Repo?

Bei jedem neuen Webprojekt schreibt man die gleichen Komponenten neu: **rechtliche Seiten**, **Cookie-Banner**, **Footer**, **Kontaktformulare**…

**dev-templates** löst dieses Problem. Es ist eine Sammlung von **sofort einsetzbaren Templates**, die in der Produktion getestet wurden und in jedes Next.js-Projekt kopiert und in wenigen Minuten angepasst werden können.

| | Vorteil |
|:-:|---------|
| ⏱️ | **Zeit sparen** — die gleichen Seiten nicht für jeden Kunden neu schreiben |
| ✅ | **Produktionsqualität** — jedes Template wurde in echten Projekten verwendet |
| 🌍 | **Open Source** — kostenlos nutzen, beitragen, mit dem Team teilen |
| 🔤 | **Platzhalter-System** — ein einfaches Suchen-und-Ersetzen passt alles an |

## 🚀 Quick Start

```bash
# 1️⃣  Repo klonen
git clone https://github.com/LuigiGarone/open-source.git

# 2️⃣  Das benötigte Template in Ihr Projekt kopieren
cp -r open-source/dev-templates/legal-pages/components/ ihr-projekt/src/components/marketing/
cp -r open-source/dev-templates/legal-pages/pages/       ihr-projekt/src/app/(marketing)/

# 3️⃣  Platzhalter ersetzen (siehe README jedes Templates)
```
